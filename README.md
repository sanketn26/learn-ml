# Applied ML Foundations for SaaS Analytics

This repository is a conversational, mentor-guided 12-week tutorial that teaches NumPy, Pandas, Visualization,
Scikit-learn, and Deep Learning through realistic SaaS analytics scenarios.

**Datasets** (generated synthetically with realistic patterns): see `data/`

- subscriptions.csv (rows: 50000)
- user_events.csv (rows: 220000)
- feature_usage.csv (rows: 160000)
- feedback.json (rows: 10000)
- product_catalog.csv (rows: 300)

Open the notebooks in `notebooks/` to start. Each week contains scenario-driven explanations, code cells, exercises, and reflection prompts.
